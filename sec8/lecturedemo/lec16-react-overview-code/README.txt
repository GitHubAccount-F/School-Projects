React demo programs for lec 16.  CSE 331, 23wi

To run these programs:

0) Download this code.  If you get the zip file, unpack it.

1) Install the current Node LTS version from https://nodejs.org/en/

2) Open this folder in intellij separately from existing projects

3) To run each demo:

	i) Open the intellij terminal (tab at the bottom of the window)
	ii) Navigate (cd) into the folder for the demo you want (hello-world, etc.)
	iii) Run npm install --noaudit      (this only needs to be done once)
	iv) Run npm start each time you want to launch the demo
